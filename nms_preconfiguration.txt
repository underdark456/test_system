Предварительная конфигурация теста сети с использованием дефолтной конфигурации (default_config.txt)

1. Получить список требуемых UHP для теста (методы get_uhp(), get_uhp_by_model() get_uhp_controllers_stations())

2. Загрузить конфигурацию в НМС: nms_api.load_config('default_config.txt')

3а. При создании контроллеров в обязательном порядке указывать параметры
    `device_ip`, `device_vlan`, `device_gateway`, `uhp_model`, `tx_level`.
	Значения параметров берутся из полученного списка доступных UHP, а не хардкодятся
    Значения параметра `tx_level` берётся из опций.

3б. При создании девайсов Smart redundancy в обязательном порядке указывать
    `ip`, `vlan`, `gateway`, `uhp_model`.
    Значения параметров берутся из полученного списка доступных UHP, а не хардкодятся

4. При создании станций значения `serial` берутся из полученного списка UHP.

5. При создании сервиса, который используется для станционного роутинга обязательно указывать параметр `stn_vlan`.
    Значение параметра берётся из полученного списка UHP.

6. При создании роута на станции типа IP_address парметр `ip` берётся из списка достпуных UHP.
    Параметр Id необходимо установить в PRIVATE.

7. При создании роута на станции типа Static_route парметр `gateway` берётся из списка достпуных UHP.
    Параметр Id необходимо установить в PRIVATE.
    Дефолтный маршрут необходимо обязательно создать даже если этого не требует локальный стенд.

8. Установить NMS permission для контроллера(ов):
    UHP драйвер инициализирован для каждого UHP, доступен по ключу `web_driver`.
	метод set_nms_permission(vlan=0, password=’’) UHP драйвера .
	Значения параметров `vlan` берётся из списка UHP, значение параметра `password` - из network

9. Установить требуемые профили на станциях:
	методы star_station(), hubless_station(), dama_station(), mesh_station() UHP драйвера.

10. Получить состояние UP для всех требуемых объектов: контроллер(ы), станции, девайсы:
	методы wait_up(), wait_state() и т.д.

11. Подождать 3 тика, чтобы станции получили конфиг: метод wait_ticks(3)
